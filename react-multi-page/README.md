# react-multi-boilerplate

A multi-page development template which base on react

## 特性

- [x] webpack4
- [x] react16
- [x] sass/less/stylus
- [x] hmr
- [x] async/await

## 如何使用

```bash
# Install dependencies
npm i

# Then start development server by running start script
npm start
```

## Production building

```bash
npm run build
```

## Simulate production server

```txt
npm run serve
```

## Run eslint

```txt
npm run eslint
```

## Unit test

```txt
npm run test:unit
```

## Full test

```txt
npm run test
```
