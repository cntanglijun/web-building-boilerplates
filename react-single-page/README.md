# react-boilerplate

A single-page development template which base on react

## Features

- [x] webpack4
- [x] react16
- [x] react-router5
- [x] sass/less/stylus
- [x] hmr
- [x] async/await

## Usage

```bash
# Install dependencies
npm i

# Then start development server by running start script
npm start
```

## Production building

```txt
npm run build
```

## Simulate production server

```txt
npm run serve
```

## Run eslint

```txt
npm run eslint
```

## Unit test

```txt
npm run test:unit
```

## Full test

```txt
npm run test
```
