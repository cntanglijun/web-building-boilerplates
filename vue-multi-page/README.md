# vue-multi-page

A multi-page development template which base on vue

## Features

- [x] webpack4
- [x] vue2
- [x] pug/ejs
- [x] sass/less/stylus
- [x] hmr
- [x] async/await

## Usage

```bash
# Install dependencies
npm i

# Then start development server by running start script
npm start
```

## Production building

```bash
npm run build
```

## Simulate production server

```txt
npm run serve
```

## Run eslint

```txt
npm run eslint
```

## Unit test

```txt
npm run test:unit
```

## Full test

```txt
npm run test
```
